kanban-md

Markdown kanban boards, served from your terminal.

A tiny Bun-powered CLI that turns any markdown file into a local Trello-style kanban board — drag, drop, and edit in the browser; every change writes back to the file.

Install from npm View on GitHub

Headings → columns

Each # heading becomes a column. Each - [ ] checkbox becomes a card.

Round-trips your markdown

The browser UI writes a canonical markdown format so diffs stay reviewable in git.

AI-friendly

Boards seed with instructions for AI assistants. Copy a one-click prompt to start implementing the Todo column.

Themes & icons

Pick from 16 preset palette colors and 40+ column icons. The whole board tints to match.

Quick start

npm install -g kanban-md
kb board.md

Requires Bun 1.3 or newer. If board.md doesn't exist, kanban-md creates a starter board.

The markdown format

# Todo
- [ ] Add CLI: serve a local board
- [ ] Polish UI: make the board feel good

# Doing
- [ ] Wire npm publish: add NPM_TOKEN in GitHub

# Done
- [x] Create markdown board: headings become columns
